🎯 Project Overview

This Inventory Management System demonstrates advanced computer science algorithms in a real-world application. It combines Consistent Hashing for distributed storage and Quadratic Probing for efficient hash table operations.

🔧 Key Technologies

  • Consistent Hashing: Distributes products across multiple warehouse nodes evenly
  • Quadratic Probing: Handles hash collisions efficiently using quadratic increments
  • Global Distribution: Tracks products across 10 major world cities
  • Real-time Analytics: Live statistics and visualizations

🌍 Features

  • Add Products: Insert products with codes, names, quantities, and global locations
  • Search & Update: Find and modify products with O(1) average time complexity
  • Delete Products: Remove products with confirmation dialogs
  • World Map: Visual representation of global product distribution
  • Analytics Dashboard: Comprehensive statistics and charts
  • Timeline Tracking: Monitor when products were added/deleted

🎓 Educational Value

This project showcases how theoretical computer science concepts are applied in modern web development. Students can learn about distributed systems, hash tables, and data structures through hands-on interaction.
